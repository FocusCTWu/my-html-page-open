## 📊 Drug Addiction Dashboard（v0.9）說明

本專案為一個以 **HTML + JavaScript** 製作的前端視覺化儀表板，用於呈現台灣藥癮統計資料，檔案名稱為 `Drug_Addiction_Dashboard_v0_9.html`。

### 🔧 技術架構

* **Tailwind CSS**：快速建構響應式 UI 與版面設計
* **Chart.js**：繪製各類統計圖表（折線圖、長條圖、圓餅圖）
* **PapaParse**：解析 CSV 資料檔
* **Lucide Icons**：提供圖示介面元素

---

### 📁 資料來源

系統預設讀取 `Drug_Addiction_Dashboard_data.csv`，若讀取失敗可手動上傳 CSV 檔案。

---

### 📊 功能特色

#### 1️⃣ 歷年趨勢分析

* 顯示各年度總人數與性別結構
* 年齡層分布（堆疊長條圖）

#### 2️⃣ 年度深度分析

* 年齡 × 性別熱力圖（Heatmap）
* 醫療資源使用情況（門診、心理治療、家族治療、職能治療）

#### 3️⃣ 成癮物質分析

* 支援切換不同物質（海洛因、安非他命、愷他命、其他）
* 顯示歷年趨勢與當年度性別比例

#### 4️⃣ 快速洞察（Quick Insights）

* 自動計算：

  * 特定物質佔比
  * 高風險年齡層
* 提供決策參考文字摘要

---

### 🔄 互動功能

* 年度下拉選單切換分析內容
* 分頁切換不同成癮物質
* 即時更新圖表與統計資訊

---

### ⚠️ 注意事項

* GitHub Pages 或本地開啟時，若 CSV 無法自動讀取，需手動上傳
* CSV 欄位名稱需符合程式預期格式（如「年度」、「總計人數」等）

---

### 🎯 專案目的

協助政策分析人員與公共衛生相關單位，快速掌握藥癮趨勢與族群特徵，作為資源配置與介入策略之依據。
